### 前提条件
数据元素**连续有序**存放。
**连续存放**即要求数据存放在数组中，不能是链表。因为链表查找需要遍历，且链表无法根据后一节点查找前一节点。
**有序存放**即要求数据有序（由大到小，或由小到大排列）。

### 代码实现
时间复杂度：O(log N)
```cpp
int BinarySearch(const vector<int> &nums, int k)
{
    if(nums.empty())
        return -1;
    int left = 0;
    int right = nums.size() - 1;
    int mid = -1;
    while(left <= right)
    {
        mid = (left + right) / 2;
        if(k < nums[mid])
            right = mid - 1;
        else if(k > nums[mid])
            left = mid + 1;
        else
            return mid;
    }
    return -1;
}
```
